# execute-wave Agent Instructions

This mold installs the execute-wave skill for Mayor orchestration of parallel agent work.

## Usage

Invoke the `execute-wave` skill when decomposing a large epic into independent issues
for parallel execution by polecats. The skill handles the full lifecycle: intake,
decomposition, parallel dispatch, convergence monitoring, and merge/close.

## Prerequisites

- Gas Town environment with `gt`, `bd` CLI tools
- GitHub CLI (`gh`) for PR management
- Mayor role with sling permissions
